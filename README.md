# 🚀 FileMerch

A full-stack digital marketplace where creators can upload, sell, purchase, and securely download digital products such as notes, templates, cheat sheets, UI kits, and other digital resources.

# 📚 What I Learned

Building FileMerch helped me understand how production-style web applications are designed and developed.

Throughout this project, I learned:

- Designing scalable MongoDB schemas
- Building RESTful APIs with Express.js
- Implementing Google OAuth authentication
- Managing authentication using JWT and HTTP-only cookies
- Creating role-based access control
- Uploading and managing files with Supabase Storage
- Generating secure signed URLs for private downloads
- Structuring a full-stack MERN application
- Building reusable React components
- Managing application state using React Context
- Protecting backend routes with middleware
- Handling file uploads and validations
- Designing marketplace workflows from seller onboarding to secure product delivery
- Deploying full-stack applications using Vercel and Render
